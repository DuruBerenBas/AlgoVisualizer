# 🧱 AlgoVisualizer: Veri Yapıları Görselleştirici

Modern veri yapılarını ve algoritmaları interaktif olarak öğrenmek ve test etmek için geliştirilmiş bir web simülasyonu aracıdır.

## 🚀 Özellikler
* **Stack (Yığın) Simülasyonu:** LIFO (Last In, First Out) mantığını görsel olarak test edebilme.
* **Hata Yönetimi (Error Handling):** Stack Overflow ve Stack Underflow durumlarının anlık olarak yakalanıp kullanıcıya bildirilmesi.
* **Modern Arayüz:** Streamlit ile geliştirilmiş, kullanıcı dostu ve hızlı web arayüzü.
* **OOP Mimarisi:** Python ile sıfırdan "Node" (Düğüm) mantığı kullanılarak yazılmış temiz backend altyapısı.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Python
* **Arayüz:** Streamlit
* **Kavramlar:** Object-Oriented Programming (OOP), Data Structures

## 💻 Kurulum ve Çalıştırma

Projeyi kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. Repoyu bilgisayarınıza klonlayın:
`git clone https://github.com/DuruBerenBas/AlgoVisualizer.git`
`cd AlgoVisualizer`

2. Gerekli kütüphaneleri yükleyin:
`pip install streamlit`

3. Uygulamayı başlatın:
`streamlit run app.py`
